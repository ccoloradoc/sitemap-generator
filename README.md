# sitemap-generator
Simple command line nodejs program to build web portal sitemap 

Commands: 
  -s    site to crawl
  -o    xml output file
  -h    help


##Initialize

```
  npm install
```

##Execute

```
  node app.js -s <site url> -o <file output>
```
